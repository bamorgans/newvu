# NewVu UI Design Specification

## Design Language

Inspired by:

- Apple Vision style glass surfaces
- Linear productivity workflows
- Video editing interfaces
- Modern AI assistants

## Primary Screens

- Login
- Dashboard
- Interview Kanban
- Live Interview Workspace
- Review Studio
- Evaluation Dashboard

## Components

- Floating glass panels
- Animated drawers
- Filmstrip timeline
- AI quote cards
- Video controls

## Interaction Principles

- Controls fade away during viewing
- Panels slide over content
- Mobile uses gestures and drawers
- Desktop uses multi-pane workspace
